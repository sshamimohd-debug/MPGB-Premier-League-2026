GitHub Pages Deploy (Quick)

  1. Upload all files of this zip to your GitHub repo root (or /docs).
  2. Repo → Settings → Pages → Source: Deploy from a branch.
  3. Select Branch: main, Folder: / (root) (or /docs).
  4. Open the Pages URL. Done.
