# MPGB Cricket Manager (GitHub Pages)

This is a **static** cricket tournament manager designed to run on **GitHub Pages** (no server required).

## Features
- Teams & Players
- Match schedule
- Toss + Two-innings live scoring (ball-by-ball)
- Scorecard + Print/PDF
- Auto points table with NRR (approx.)
- Admin PIN protection (set in Admin page)
- Offline-ready (PWA) + Export/Import JSON

## Deploy to GitHub Pages
1. Create a GitHub repo (public or private).
2. Upload all files from this folder to repo root.
3. Repo → **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`
4. Open the provided Pages URL.

## Local run
Just open `index.html` in browser (best via a local web server for service worker).
